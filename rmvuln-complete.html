OSI Model Overview

The Open Systems Interconnection (OSI) model is a conceptual framework that standardizes the functions of a communication system into seven distinct layers.

Layer # Name Function Common Protocols
7 Application User interfaces, apps HTTP HTTPS FTP SMTP DNS DHCP SNMP Telnet SSH
6 Presentation Data formatting, encryption SSL/TLS JPEG MPEG GIF ASCII EBCDIC
5 Session Connection management NetBIOS PPTP RPC SMB NFS
4 Transport End-to-end communication TCP UDP SCTP DCCP
3 Network Routing, addressing IP IPv4 IPv6 ICMP IPsec IGMP ARP
2 Data Link Node-to-node transfer Ethernet WiFi (802.11) ARP PPP HDLC MAC
1 Physical Physical transmission Ethernet (Physical) USB Bluetooth DSL ISDN RS-232
